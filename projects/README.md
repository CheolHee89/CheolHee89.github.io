# 프로젝트 PDF 파일

이 폴더에는 프로젝트 섹션에서 다운로드할 수 있는 PDF 파일들을 저장합니다.

## 파일 구조

다음과 같은 파일명으로 PDF를 저장해주세요:

- `project1.pdf` - 첫 번째 프로젝트 PDF
- `project2.pdf` - 두 번째 프로젝트 PDF
- `project3.pdf` - 세 번째 프로젝트 PDF

## 사용 방법

1. 각 프로젝트에 해당하는 PDF 파일을 준비합니다.
2. 파일명을 위의 형식에 맞게 변경합니다.
3. 이 폴더에 파일을 저장합니다.
4. 웹사이트의 프로젝트 섹션에서 PDF 다운로드 버튼이 자동으로 작동합니다.

## 참고사항

- PDF 파일은 최대 10MB 이하로 권장합니다.
- 파일명은 소문자로 작성하는 것을 권장합니다.
- PDF 파일이 없어도 웹사이트는 정상적으로 작동하지만, 다운로드 시 오류가 발생할 수 있습니다.
